\# Insurance Risk Portfolio Analysis



\## Overview

This project analyzes an insurance portfolio using quantitative risk modeling techniques. 

It calculates key risk metrics and visualizes loss distributions to support decision-making.



\## Objectives

\- Compute Total Expected Loss

\- Calculate Value at Risk (VaR)

\- Calculate Conditional Value at Risk (CVaR)

\- Perform Monte Carlo simulation for portfolio risk



\## Project Structure

\- data/  

&#x20; Contains input datasets and generated graphs  



\- src/  

&#x20; Python script for data generation and analysis  



\- notebooks/  

&#x20; Jupyter notebook used for portfolio analysis  



\- report/  

&#x20; Final LaTeX report and compiled PDF  



\- requirements.txt  

&#x20; List of Python dependencies  



\## Key Results

\- Total Expected Loss: 2,528,305.57  

\- 95% VaR: 7,119.42  

\- 95% CVaR: 8,572.66  

\- Monte Carlo 95% VaR: 2,981,595.75  

\- Monte Carlo 99% VaR: 3,177,618.88  



\## How to Run

1\. Install dependencies:

&#x20;  pip install -r requirements.txt



2\. Run the Python script:

&#x20;  python src/data\_generation.py



3\. Open the notebook:

&#x20;  notebooks/01\_portfolio\_analysis.ipynb
