# Insurance Portfolio Risk & Valuation Analysis

## Overview
This project demonstrates a quantitative analysis of a simulated insurance portfolio. 
It includes data generation, exploratory data analysis, frequency-severity modeling, 
risk metrics calculation, and stress testing.

## Project Structure
- `data/` → raw and processed datasets
- `notebooks/` → Jupyter notebooks for each step of analysis
- `src/` → Python scripts for functions and models
- `excel/` → Excel summaries
- `reports/` → PDF reports

## How to Run
1. Install Python packages: `pip install -r requirements.txt`
2. Open notebooks in order: `01_data_generation.ipynb → 05_stress_testing.ipynb`
